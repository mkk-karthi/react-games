# 🧩 Puzzle Tetris

A modern **Tetris-style puzzle game** built using **React + TypeScript** with **Vite** and styled with **Tailwind CSS**.  
The game includes smooth animations, sound effects, and full mobile support for an enjoyable experience across devices.

---

## 🚀 Features

- 🎵 Sound effects for moves, line clears, and game over  
- 🎬 Smooth animations for block movement and line clearing  
- 📱 Fully responsive and mobile-friendly controls  
- ⌨️ Keyboard support for desktop users  
- ⚡ Fast and optimized performance  

---

## 🛠️ Tech Stack

- ⚛️ **React** (with TypeScript)
- 🎨 **Tailwind CSS**
- 🔊 **Sound Effects**
- 🎞️ **CSS / Animation Effects**
- 📱 **Mobile Responsive Design**

---

## 🎮 Controls

### Desktop
- ⬅️ / ➡️ Arrow Keys – Move block left/right  
- ⬇️ Arrow Key – Soft drop  
- ⬆️ Arrow Key – Rotate block  
- Space – Hard drop  

### Mobile
- 👆 Tap / Swipe gestures for movement and rotation  
- 📲 Optimized touch controls

## 🧑‍💻 Getting Started

### Prerequisites

- Node.js & npm installed

### Installation

### Clone the repository

```bash
git clone https://github.com/mkk-karthi/react-games.git
cd react-games/puzzle-tetris

```

### Install dependencies

```bash
npm install
```

### Run locally

```bash
npm run dev
```

The app will run at `http://localhost:3000`.
